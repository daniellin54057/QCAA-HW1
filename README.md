This repository contains the implementation code for HW1 of the QCAA class.
For Problem 1, I explored several data-reuploading quantum circuit architectures with different numbers of layers and qubits, compared their performance, plotted their predictions, and analyzed the Fourier spectra of the target function and model output.
For Problem 2, I reproduced the performance comparison on the circle dataset, generated decision-boundary plots for three methods on both the circle and moons datasets, and compared them in terms of accuracy, complexity, and training time.
